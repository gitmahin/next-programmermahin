# HTML Tutorial (Basic to Advanced)

---

## Introduction to HTML
HTML (HyperText Markup Language) is the standard language for creating web pages.
```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
```
---

# HTML Basic Tags


### Paragraphs


## Advanced Concepts
### Custom Data Attributes
```html
<!-- [!code word:data-user-id] -->
<div data-user-id="1234">User Info</div>
```

### HTML5 Input Validation
```html
<input type="email" required>
<input type="text" pattern="[A-Za-z]{3,}">
```

### Form Autofill & Autocomplete
```html
<input name="email" autocomplete="email">
```
---

## Accessibility
```html
<img src="img.jpg" alt="Descriptive text">
<label for="email">Email</label>
<input id="email" type="email">
```
Use `alt`, `label`, `role`, and semantic tags to improve accessibility.

---

## Best Practices
- Use semantic HTML.
- Keep structure clean.
- Use comments to explain code.
```html
<!-- This is a comment -->
```

## HTML + CSS + JS Integration
```html
<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="style.css"> <!-- [!code highlight] -->
  </head>
  <body>
    <h1>Hello</h1>
    <script src="script.js"></script> <!-- [!code highlight] -->
  </body>
</html>
```
